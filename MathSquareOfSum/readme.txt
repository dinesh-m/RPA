It is a C# class library project.
The project references System.Activities & System.ComponentModel.Composition assemblies.
Class 'SimpleFormula' inherits from CodeActivty abstract class, which has an Execute method.
Three field methods are created- two for inputs and one for output.
The project builds into a *.dll, which is used to create a nuget package.
The nuget package 'ActivitiesMathSquare.1.0.3.nupkg' is contained inside SumSquare folder.
